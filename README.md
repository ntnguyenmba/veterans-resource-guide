# DAV Chapter 246 Veterans Resource Guide

A polished, print-friendly HTML guide for veterans, service members, families, caregivers, and survivors in Tarrant County, Texas.

## Files

- `index.html` contains the full guide, styling, QR code directory, map links, print button, and responsive layout.

## Image slots

The guide is ready to use client-provided photos. Add these files inside an `assets` folder when available:

- `assets/cover-veterans.jpg`
- `assets/veterans-community.jpg`
- `assets/dav-service-officer.jpg`

If the images are not added yet, the page still displays with designed color overlays and fallback styling.

## Final review before sending to client

Before publishing the final PDF or live page, verify current phone numbers, office hours, addresses, and eligibility rules directly with the official agencies because veterans programs can change.

## Suggested next step

Deploy this repository to Netlify or GitHub Pages, then print the page to PDF for a polished handout version.
